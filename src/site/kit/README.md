# Lab Pages kit

This folder contains reusable Astro primitives for Simon Gelbart project documentation sites.

It is currently internal to Pattrn while the project-site model stabilizes.

Reusable responsibilities:

- docs manifest reading
- source-link generation
- internal/external link handling
- docs navigation
- rendered Markdown document layout
- base visual primitives shared by project docs sites

Not included:

- project-specific landing copy
- project-specific package descriptions
- project-specific roadmap or benchmark wording
- main-site editorial project data

Future extraction:
If LORQ or dotnet-learning-lab adopts this model, this folder can be moved into a shared package or standalone repository after at least two project sites use it.
